h1. Manipulating GSimpleNtpFlux files

h2. What&#39;s in the file?

One can use the command:<pre>   $ genie -b -q &#39;dump_gsimple_file.C("gsimple_input.root")&#39;</pre> e.g. <pre>   $ genie -b -q $GSIMPLE/scripts/dump_gsimple_file.C\(\"/nova/data/flux/gsimple/nova_v03/ndos/mn/fhc/*q00*.root\"\)</pre> (actual script also available below, but also available to all in @~rhatcher/public@) to dump:
* the configuration of the GSimpleNtpFlux when initialized with the _gsimple_input.root_ file
* a handful of flux rays
* the associated metadata

h3. Individual flux ray entries

Individual rays have 3 branches associated with them on the tree.  The basic _entry_ branch contains the minimal information to define a neutrino ray.  The _aux_ branch is a means of extending the ntuple with extra information without modifying the compile class code.  This branch consists of a vector of ints and doubles; to interpret the meaning of each element in the arrays the names should bin in corresponding entries in the metadata (in this example the two doubles represent the @fgXYWgt@ and @nimpwt@ values).  The (optional) third branch, _numi_, carries information that has been used in previous efforts for beam reweighting and information that should help tie this ray back to the original @flugg@ or @g4numi@ ntuple entry.
<pre>

h3. Quick peek

Alternatively, you can do a quick peek at the meta data by hand with:
<pre>   $ genie gsimple_input.root</pre>and then at the genie prompt type:
<pre>
   genie::flux::GSimpleNtpMeta* metaobj = new genie::flux::GSimpleNtpMeta;
   meta->SetBranchAddress("meta",&amp;metaobj);
   meta->GetEntry(0);
   metaobj->Print();
</pre>

h2. Creating new GSimpleNtpFlux files by modifying an existing one

One can create a new flux file by modifying entries between reading an entry and writing it to a new file.
<pre>  $ genie -b -q &#39;warp_gsimple.C+("gsimple_out.root","gsimple_in.root")&#39;</pre>

The user needs to modify the script&#39;s @warp_entry()@ and @warp_meta()@ functions to do whatever manipulation of the entries they desire.
